# 지금 몇교시야

현재 교시, 다음 일정까지 남은 시간, 개인 시간표를 한눈에 보여주는 데스크톱 위젯입니다.

![메인 화면 안내](docs/images/main-controls.svg)

## 다운로드

- [최신 Windows / macOS 릴리즈](https://github.com/RamzThunder/whattime-releases/releases/latest)
- Windows: `whattime.exe`
- macOS: `WhatTime-mac.dmg`

## 빠른 시작

1. 운영체제에 맞는 설치 파일을 내려받습니다.
2. 앱을 실행하고 오른쪽 위의 `설정` 버튼을 누릅니다.
3. `일정 관리`에서 일반 시정과 개인 시간표를 입력합니다.
4. `저장`을 누르면 메인 위젯에 바로 반영됩니다.

## 주요 기능

- 현재 교시와 다음 일정까지 남은 시간 표시
- 일반 시정, 단축 시정, 쉬는 날 일정 관리
- 요일별 개인 시간표와 교실 표시
- 컴시간 알리미 시간표 불러오기
- 미니 모드, 항상 위 고정, 배경 표시 전환
- 색상, 글꼴, 글자 크기, 밀리초 표시 설정
- 부팅 시 자동 실행과 자동 업데이트
- 설정 백업, 복원, 기본값 지정

상세 사용법은 [사용설명서](docs/USER_GUIDE.md)를 참고하세요.

## 문제 해결

- 앱이 보이지 않으면 작업 관리자에서 `whattime.exe`를 종료한 후 다시 실행하세요.
- Windows에서는 네트워크 드라이브보다 로컬 폴더에서 실행하는 것을 권장합니다.
- 자동 업데이트가 실패하면 최신 `whattime.exe`를 직접 내려받아 교체하세요.
